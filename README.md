# extract_rooms_v2

Сервис для администраторов студсовета ННГУ: автоматизация подбора свободных аудиторий и генерация служебных записок (PDF) через Telegram-бота.

## Возможности
- **Интерактивный Telegram-бот:** Многорежимный интерфейс для поиска свободных аудиторий и формирования документов.
- **Генерация служебных записок (PDF):** Автоматическое создание файлов по встроенным шаблонам (Расписание, Холл, Освобождение, Внос-Вынос).
- **Подбор аудиторий:** Умный поиск помещений для пакета запросов без пересечений внутри одного батча. Поддержка различных типов вместимости (`any`, `big`, `big2`, конкретный номер и т.д.).
- **Фоновое обновление данных:** Скачивание raw JSON по корпусам из `/ruzapi/schedule/building/{building_oid}`, обрезка до минимального формата и кэширование в in-memory репозитории.
- **CLI-инструменты:** Различные режимы запуска для тестирования аллокации, генерации PDF и загрузки расписания без запуска бота.

## Формат входного запроса (для поиска аудиторий)
В режиме планирования бот и CLI принимают строки в следующем формате:
`Имя Фамилия Цель дд.мм чч:мм чч:мм тип_аудитории`

Поддерживаемые типы аудитории:
- `any`, `any2`, `any6`
- `big`, `big2`, `big6`
- Конкретный номер аудитории (например, `140` или `105 (корпус 2)`)
